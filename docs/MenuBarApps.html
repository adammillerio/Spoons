docs » MenuBarApps

Control applications from the macOS Menu Bar

Download: https://github.com/adammillerio/Spoons/raw/main/Spoons/MenuBarApps.spoon.zip

Example Usage (Using SpoonInstall): Create a "P" menu bar item that opens and moves Plexamp and create a "D" menu bar item that opens and maximizes Discord. spoon.SpoonInstall:andUse( "MenuBarApps", { config = { apps = { ["Plexamp"] = { title = "P", action = "move" }, ["Discord"] = { title = "D", action = "maximize" } } }, start = true } )

API Overview

  • Constants - Useful values which cannot be changed
    • maximize
    • move
  • Variables - Configurable values
    • apps
    • logger
    • logLevel
    • menuBars
  • Methods - API calls which can only be made on an object returned by a constructor
    • init
    • start
    • stop
