docs » Spacer

Name and switch Mission Control spaces in the menu bar

Download: https://github.com/adammillerio/Spoons/raw/main/Spoons/Spacer.spoon.zip

Example Usage (Using SpoonInstall): spoon.SpoonInstall:andUse( "Spacer", { start = true } )

Space names can be changed from the menubar by holding Alt while selecting the desired space to rename. These are persisted between launches via the hs.settings module.

API Overview

  • Constants - Useful values which cannot be changed
    • settingsKey
  • Variables - Configurable values
    • logger
    • logLevel
    • menuBar
    • orderedSpaceNames
    • orderedSpaces
    • spaceNames
    • spaceWatcher
  • Methods - API calls which can only be made on an object returned by a constructor
    • init
    • start
    • stop
